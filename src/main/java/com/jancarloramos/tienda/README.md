# 🛒 Tienda Spring Boot - Gestión de Usuarios y Pedidos

Este proyecto es una aplicación web desarrollada con **Spring Boot** y **Thymeleaf** para gestionar el registro de usuarios y los detalles de pedidos. Cumple con los requisitos de validación de datos tanto en el frontend como en el backend.

##  Guía de Estilo (Paleta de Colores)

| Color | Nombre | Código HEX | Significado / Uso |
| :--- | :--- | :--- | :--- |
| Azul Marino | Navy Blue | `#1a237e` | Color principal para encabezados, barras de navegación y elementos estructurales importantes. |
| Morado Oscuro | Deep Purple | `#4a148c` | Fondo secundario o secciones destacadas; aporta profundidad y contraste. |
| Morado Claro | Light Purple | `#7c43bd` | Acentos visuales, hover en botones o elementos interactivos. |
| Blanco | White | `#ffffff` | Texto principal sobre fondos oscuros y áreas limpias para mejorar la legibilidad. |
| Gris Fondo | Light Gray | `#f0f2f5` | Fondo general de la aplicación para mantener una apariencia moderna y suave. |
